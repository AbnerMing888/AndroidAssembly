# AndroidAssembly

AndroidAssembly是一个Android组件化相关Demo，列举了常见的组件化功能实现。 

绝对是全网最简单、最详细的的组件化学习代码，手把手教你学习、了解，掌握组件化开发。

大家可对着文档进行系统的学习，文章地址如下：

[组件化对应文章学习](https://mp.weixin.qq.com/s/eLGEWvHtyqtvrtScr6Doog)
